# JavaScript - Warm Up

## Description
This project is an introduction to JavaScript programming language. It covers basic concepts such as variables, data types, control flow, functions, and more.

## Learning Objectives
- Understand why JavaScript programming is amazing
- Learn how to run a JavaScript script
- Create variables and constants
- Differentiate between `var`, `const` and `let`
- Use all data types available in JavaScript
- Implement `if`, `if ... else` statements
- Use comments effectively
- Work with `while` and `for` loops
- Use `break` and `continue` statements
- Understand functions and their usage
- Learn about variable scope
- Use arithmetic operators
- Manipulate objects (dictionaries)
- Import files in JavaScript

## Requirements
- All files will be interpreted on Ubuntu 20.04 LTS using `node` (version 14.x)
- All files should end with a new line
- The first line of all files should be exactly `#!/usr/bin/node`
- Code should be `semistandard` compliant (version 16.x.x)
- All files must be executable

## Installation
```bash
# Install Node.js 14
curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
sudo apt-get install -y nodejs

# Install semistandard for code style checking
sudo npm install semistandard --global
