# Python Rectangle Class
A series of tasks implementing a Rectangle class with progressively advanced features.

## Tasks
- 0-rectangle.py: Empty Rectangle class
- 1-rectangle.py: Adds width/height with validation
- 2-rectangle.py: Adds area and perimeter methods
- 3-rectangle.py: Adds string representation (# characters)
- 4-rectangle.py: Adds eval() compatible representation
- 5-rectangle.py: Adds deletion message
- 6-rectangle.py: Tracks instance count
- 7-rectangle.py: Customizable print symbol
- 8-rectangle.py: Rectangle comparison by area
- 9-rectangle.py: Adds square constructor
