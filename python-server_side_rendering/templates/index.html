Welcome to My Flask App

This is a simple Flask application demonstrating the power of server-side rendering with Jinja2 templates.

Features of this application:

  • Flask web framework
  • HTML templates with Jinja2
  • Template inheritance and includes
  • Reusable components
  • Dynamic content with loops and conditions
  • JSON data integration

Pages:

  • Home - This page
  • About - Information about the app
  • Contact - Contact information
  • Items - Dynamic items from JSON (NEW!)

Technologies Used:

  • Python 3.x
  • Flask framework
  • HTML5
  • CSS3
  • Jinja2 templating engine
  • JSON for data storage

Visit the Items page to see dynamic content with loops and conditions!
