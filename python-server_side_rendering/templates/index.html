Welcome to My Flask App

This is a simple Flask application demonstrating the power of server-side rendering with Jinja2 templates.

Features of this application:

  • Flask web framework
  • HTML templates with Jinja2
  • Template inheritance and includes
  • Reusable components
  • Responsive design
  • Clean and maintainable code structure

Technologies Used:

  • Python 3.x
  • Flask framework
  • HTML5
  • CSS3
  • Jinja2 templating engine

This application serves multiple pages with shared header and footer components, demonstrating best practices in web development.
