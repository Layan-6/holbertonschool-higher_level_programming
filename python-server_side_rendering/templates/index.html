Welcome to My Flask App

This is a Flask application demonstrating server-side rendering with Jinja2 templates.

Features:

  • Flask web framework
  • HTML templates with Jinja2
  • Dynamic content with loops and conditions
  • JSON and CSV data integration
  • Query parameter handling
  • Error handling for edge cases

Available Pages:

  • Home - This page
  • About - Information about the app
  • Contact - Contact information
  • Items - Dynamic items from JSON
  • Products (JSON) - Products from JSON file
  • Products (CSV) - Products from CSV file

Test Different Scenarios:

  1. JSON all products
  2. CSV all products
  3. JSON product ID 3
  4. CSV product ID 5
  5. Invalid source (error)
  6. Non-existent ID (error)
