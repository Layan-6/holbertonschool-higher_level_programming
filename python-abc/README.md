# Python OOP Learning Exercises
A structured exploration of Object-Oriented Programming (OOP) concepts in Python through practical implementation exercises.

## Exercises
1. Abstract Animal Class – Implement an abstract base class for animals and create concrete animal implementations
2. Shapes & Duck Typing – Create geometric shapes using duck typing without explicit interface checking
3. Smart List – Build a list class that provides notifications on state changes
4. Counted Iterator – Implement an iterator that tracks iteration count
5. FlyingFish Multiple Inheritance – Explore multiple inheritance through a creature combining fish and bird characteristics
6. Mystical Dragon with Mixins – Use mixins to add specialized abilities to a dragon class
