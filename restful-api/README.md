# Task 0: Basics of HTTP/HTTPS

## 1. Differences between HTTP and HTTPS

### HTTP
- Unsecured protocol
- Data sent as plain text
- Uses port 80
- No SSL/TLS certificate required
- Vulnerable to eavesdropping

### HTTPS
- Secure protocol
- Data encrypted with SSL/TLS
- Uses port 443
- Requires SSL/TLS certificate
- Protects data from interception
